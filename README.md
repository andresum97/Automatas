# Automatas
Proyecto #1 de la clase de Lenguajes de Programación

## ¿Como correrlo?

Para correr el programa se debe ejecutar el archivo **lecturacocol.py**

El ATG a utilizar se configura en la linea 508. Este se selecciona dentro del parametro de la clase **Lectura**

Ejemplo
> lec = Lectura('CoCoL.ATG')


Este genera el archivo scanner.py el cual se corre por separado para ver el scanner funcionando. 
### Importante
Para poder probar el scanner es necesario tener un archivo de texto llamado **palabras.txt** y dentro de este archivo deben ir las palabras que se quieren probar
con los tokens leidos en el archivo de Cocol.
